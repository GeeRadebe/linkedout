# LinkedOut: Local-First Career Management Suite

LinkedOut is a specialized desktop application built with Electron.js designed for high-performance job application tracking and career telemetry.

## 🚀 Key Features
- **Local-First Architecture:** Ensures data privacy and ultra-low latency without reliance on external cloud storage.
- **Application Lifecycle Tracking:** Manage the full pipeline from 'Applied' to 'Contract' with automated status updates.
- **Document Management:** Centralized hub for tailoring CVs and cover letters per application.

## 🛠️ Tech Stack
- **Framework:** Electron.js (Cross-platform Desktop)
- **Frontend:** HTML5 / CSS3 / JavaScript
- **Data Layer:** SQLite / LocalStorage for offline persistence.
